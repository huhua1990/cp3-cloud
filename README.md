# cp3-cloud

## 简介
[cp3-cloud](https://gitee.com/huhua1990/cp3-cloud) 是打造一套兼顾 SpringBoot 和 SpringCloud 项目的公共工具类。

## 结构介绍
![cp3-cloud-commons功能介绍.png](docs/images/cp3-cloud-commons功能介绍.png)

## 主要模块功能
- cp3-cloud-comon   核心工具模块
- cp3-oauth      权限模块
- cp3-authority   认证模块
- cp3-tenant  租户模块
- cp3-jobs  任务调度模块
- cp3-file  附件模块
- cp3-msgs  消息模块
- cp3-support  支持相关模块
