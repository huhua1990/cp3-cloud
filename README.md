# cp3-cloud

## 仅供个人学习使用

## 简介
[cp3-cloud](https://github.com/huhua1990/cp3-cloud) 是以springcloud为基础打造的全方面微服务应用框架。

## 主要模块功能
- cp3-base   核心工具模块
- cp3-oauth      权限模块
- cp3-authority   认证模块
- cp3-tenant  租户模块
- cp3-jobs  任务调度模块
- cp3-file  附件模块
- cp3-msgs  消息模块
- cp3-activiti activti工作流模块
- cp3-activiti-flowable flowable工作流模块
